# BruteGmail
Ini adalah Skrip Python3 untuk meretas akun Gmail menggunakan Metode Dictionary Attack.

# penggunaan

**Cloning Repository:**
```
git clone https://github.com/Yoga913/BruteGmail.git
```
**Masuk Direktory:**
```
cd BruteGmail
```
**Jalankan:**
```
Python3 BruteGmail.py
```
